# Insight UI

## Install & Start
You need to be running [the latest node LTS](https://nodejs.org/en/download/) or newer.

```bash
git clone https://github.com/bitpay/insight-ui.git
cd insight-ui/app
npm install
npm start
```

## Run Unit Tests
```bash
npm test
```

## Run E2E
```
npm run e2e
```
